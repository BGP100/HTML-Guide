The <b>&lt;br&gt;</b> tag inserts a single line break.
<br>
The <b>&lt;br&gt;</b> tag is useful for writing addresses or poems.
<br>
The <b>&lt;br&gt;</b> tag is an empty tag which means that it has no end tag.
