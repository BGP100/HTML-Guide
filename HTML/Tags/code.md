The <b>&lt;code&gt;</b> tag is used to define a piece of computer code. The content inside is displayed in the browser's default monospace font.
<br>
<b>Tip:</b> This tag is not deprecated. However, it is possible to achieve richer effect by using CSS (see example below).

<h1>Default CSS</h1>
Most browsers will display the <b>&lt;code&gt;</b> element with the following default values:
<pre>
code {
  font-family: monospace;
}
</pre>
