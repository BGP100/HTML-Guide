HTML5 is the same thing as HTML but it has these new things:
<ol>
  <li>&lt;article&gt;</li>
  <li>&lt;aside&gt;</li>
  <li>&lt;footer&gt;</li>
  <li>&lt;header&gt;</li>
  <li>&lt;main&gt;</li>
  <li>&lt;nav&gt;</li>
  <li>&lt;section&gt;</li>
</ol>
<b>NOTE:</b> If you are using HTML5 always put <code>&lt;html xmlns="//www.w3.org/1999/xhtml"&gt;</code>
