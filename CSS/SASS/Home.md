SASS is a CSS pre-processor.
<br>
SASS reduces repetition of CSS and therefore saves time.
<h1>Example</h1>
<pre>
/* Define standard variables and values for website */
$bgcolor: lightblue;
$textcolor: darkblue;
$fontsize: 18px;
/* Use the variables */
body {
  background-color: $bgcolor;
  color: $textcolor;
  font-size: $fontsize;
}
</pre>
