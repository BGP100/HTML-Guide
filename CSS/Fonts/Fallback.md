<h1>Commonly Used Font Fallbacks</h1>
Below are some commonly used font fallbacks, organized by the 5 generic font families:
<ul>
  <li><b>Serif</b></li>
  <li><b>Sans-serif</b></li>
  <li><b>Monospace</b></li>
  <li><b>Cursive</b></li>
  <li><b>Fantasy</b></li>
</ul>
<h2>Serif Fonts</h2>

  <tr>
